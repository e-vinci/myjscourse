Vitrine de projets : Web 2020

Voici les projets qui ont été développés et rendus publics par les étudiants du bloc 2 en Informatique de gestion à Vinci.

Projet Groupe N°4
Description
Jeu en 2D de touche-touche en 1 contre 1. Jeu multijoueur en local où un joueur incarnera un chat qui poursuit l'autre joueur incarné par une poule. Chaque joueur possède 3 vies à chaque début de partie. Le but du jeu est d'attraper son adversaire (la poule ou le chat), ou d'en fuir, dépendant du joueur que l'on incarne. Des pièges seront posés, ainsi que des obstacles, qui compliqueront le jeu. Il y aura également des bonus et des malus, rendant le jeu encore plus dingue ! Le jeu s arrête quand l'un des deux joueurs n a plus de vie. Il est important de noter que le chasseur et le chassé pourront échanger les rôles tous les X temps, afin de rajouter du peps/dynamisme.
Vidéo de présentation

Repo backend

Projet Groupe N°2
Description
Game Items Collection est un site web qui permet à chaque utilisateur de lister les objets virtuels qu’il a acquis dans les divers jeux auxquels il a joué. Le site permettra aussi d’effectuer une recherche afin de trouver l’objet qui vous intéresse en et de voir les personnes qui détiennent cet objet. Une fois l’objet qui vous intéresse trouvé, il sera aussi possible de communiquer auprès d’un possesseur afin de voir si il est possible de le lui acheter ou d’effectuer un échange d’objets.
